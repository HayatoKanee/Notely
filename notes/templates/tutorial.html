Tutorial


How to start our website

  1. Find the (in your upper right corner),
    If you do not have an account, please go to the page.

  2. After logging in, you will enter the notebook home page.

In the notebook home page

  • Add folder / notebook.
    1. You will see the "New" button, click on it.
    2. Then you will find a pop-up window indicating whether to add a "Folder" or a "Notebook".
    3. Type Folder name / Notebook name.
  • folder options / notebook options.
    1. You can see there is delete to delete folder or notebook.
    2. Also you can link pages.

In the note page

  • Pen button
    1. Select color
    2. Select pen style
  • Text button
    1. Select text color
    2. Text filed
  • Table button
    1. Create table
    2. Resize the table
  • Codeditor button
    1. Plus button
  • Image button
    1. Select file
  • Download button
    1. Download prompt
  • Notebook sidebar
    1. Add pages
    2. Dropdown menu

In the calendar

  1. You will see:

    • Monthly calendar
    • Weekly calendar
    • Day calendar
    • List

  2. How to create new event

    1. Click the "Add New Event" in your lower left corner. Then you will see a form pop up.
    2. Write down the details you want to record.
    3. Selecting your date and time.
    4. Add tag (If you don't have a tag on the form, go to "How to create a tag")
    5. Choose page
    6. Set a reminder
    7. Finally, press the "Create event" button

  3. How to create new tag

    1. Click the "Add New Event".
    2. Scroll to the bottom of the form, click the "New tag".
    3. Then you will see a form for adding tags.
    4. Fill in the title and color you want.
    5. click the "Create tag".

  4. After you create an event

    • you can find events on each calendar.

    • Clicking an event on the calendar will pop up the details of the event.

      • You can click "Edit" to modify the file.
      • Delete the event.

  5. How to share event

    1. Click the "Share event" in your lower left corner. Then you will see a form pop up.
    2. Write down the email of the person you want to send and the message you want to write.
    3. Click the "Share event".

Go to profile

  • You can fill in your personal information on the profile and make changes at any time. Just click "Edit".

  • If you would like to change your profile picture.

    1. Go to the upper right corner and click your profile.
    2. Choose "Change gravatar"
    3. Then you will be redirected to a web page.
    4. Click "Sign in". (Please make sure your registered email is the same as Notely email)

  • If you would like to change your password.

    1. Go to the upper right corner and click your profile.
    2. Choose "Change password"
